## About
chtitle changes the title of xterminals.

## Usage
chtitle.sh : a title changer
Usage
	 chtitle.sh -h : display usage.
	 chtitle.sh -t [title] : set the title
	 chtitle.sh -c         : use the current folder name as title.
	 chtitle.sh -f         : ignore the terminal type when setting title.
